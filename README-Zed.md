# Blank Theme for Zed

![Blank Theme Zed](zed.png)

An Elegant and Minimalist Theme for Zed, ported from VSCode.

## Themes Included

- **Blank Moonlight** (Dark, Purple/Blue/Orange accents)
- **Blank Monochrome** (High contrast, Black/White)
- **Blank Ghibli** (Soft, earth tones)
- **Blank Uchiha** (Dark, Red accents)

## Installation & Usage

You can install this theme locally by cloning the repository and loading it into Zed as a Dev Extension.

### 1. Clone the Repository

Open your terminal and clone this repository to a location of your choice:

```bash
git clone https://github.com/kiritocode1/Blank-Theme.git
cd Blank-Theme
```

### 2. Install in Zed

1. Open **Zed**.
2. Press `Cmd+Shift+P` (macOS) or `Ctrl+Shift+P` to open the command palette.
3. Search for and select: **`zed: extensions`**
4. Click on **Install Dev Extension** in the top right corner.
5. Navigate to the `Blank-Theme` folder you just cloned and select it.

The theme should now be installed! You can select your preferred flavor via the theme selector (`Cmd+K Cmd+T` or `Ctrl+K Ctrl+T`).

## Development & Customization

If you want to modify the themes, you can edit the `.json` files in the `themes/` directory directly. Changes should reflect immediately or after a quick window reload.
