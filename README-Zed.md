# Blank Theme for Zed

An Elegant and Minimalist Theme for Zed.

## Themes Included
- Blank Moonlight
- Blank Monochrome
- Blank Ghibli
- Blank Uchiha

## Installation

1.  **Direct Download**:
    - Download the `.json` files from `themes/`.
    - Place them in `~/.config/zed/themes/` (macOS/Linux).
    - Restart Zed.

2.  **Extension**:
    - Build and install this extension in Zed's developer mode.
    - Or find it in the Zed extension registry (coming soon).
